# How to Run and Test

## Step 1: Run the program
In terminal, type exactly:
```
C:\flutter\bin\dart run lib\main.dart
```
Press Enter.

## Step 2: Enter file path
When you see "File path:" - type ONLY ONE of these:

---

### Test File 1 - Valid Students (A,B,C,D,F grades):
```
test_data/valid_students.xlsx
```

### Test File 2 - Empty File:
```
test_data/empty_file.xlsx
```

### Test File 3 - Missing Names:
```
test_data/missing_names.xlsx
```

### Test File 4 - Invalid Scores:
```
test_data/invalid_scores.xlsx
```

### Test File 5 - Mixed Data (all edge cases):
```
test_data/mixed_data.xlsx
```

---

## Important:
- Type the path WITHOUT "File path:" prefix
- Use FORWARD SLASHES "/" not backslashes "\"
- Press Enter after typing the path
